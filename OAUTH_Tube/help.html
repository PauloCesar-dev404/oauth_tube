Sobre a Extensão

A extensão oauth_tube automatiza a captura das credenciais necessárias para usar em aplicações ou automações.

Como Usar:

Após inicializar a API local oauth_tube em seu ambiente virtual, execute o comando:

oauth_tube init

Depois, você pode abrir a página do YouTube e a extensão irá capturar os dados necessários para autenticação.

Atenção!

Se você estiver utilizando redes não confiáveis, esteja ciente de que a extensão transmite dados pela rede. Por questões de segurança, é altamente recomendável não utilizar a extensão em redes públicas ou compartilhadas, pois suas credenciais podem ser capturadas.

Em caso de dúvida sobre a segurança do ambiente, recomendamos não utilizar a extensão oauth_tube até garantir que a rede seja segura.

Problemas Comuns

Se você encontrar problemas ao tentar usar a extensão, verifique os seguintes pontos:

  • Certifique-se de que a API local oauth_tube está rodando corretamente.
  • Verifique se a URL da API está acessível no seu navegador (http://127.0.0.1:4474/api/status).
