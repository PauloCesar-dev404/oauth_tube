Esta extensão automatiza a captura das credenciais necessárias para utilizar em modo usuário autenticado na lib youtubeiPy.

Para utilizá-la é necessário iniciar a API local oauth_tube

Após baixar, instale usando: 

                
                    pip install oauth_tube-1.0-py3-none-any.whl

Após a instalação em seu ambiente virtual, você pode chamar o comando: 

                
                    oauth_tube init
